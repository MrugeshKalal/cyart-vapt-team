# Week 3 - Advanced VAPT & Web Application Security Testing

## Overview

This repository contains the practical and theoretical activities completed during Week 3 of the Vulnerability Assessment and Penetration Testing (VAPT) training program.

The focus of this week was on:

* Advanced exploitation techniques
* Web application security testing
* Vulnerability reporting
* Post-exploitation evidence collection
* Capstone project execution
* Professional documentation practices
